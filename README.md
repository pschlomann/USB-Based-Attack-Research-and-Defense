# USB-Based-Attack-Research-and-Defense
GitHub repository containing research on USB Security. Each folder contains payloads and write-ups for different USB attacks. These attacks consist of Digispark and DuckyScript keystroke injection payloads and malware delivery through USB drive. Techniques like backdoors, reverse shells, and obfuscation are employed. These attacks were made to target a Kali Linux virtual machine or Windows Server 2008 R2 virtual machine. Additionally, fliers, a security awareness quiz, comprehensive policy, IT implementation guide, presentation, and full technical report are included in the root of the repository.
